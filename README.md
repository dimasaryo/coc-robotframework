# coc-robotframework

coc.nvim extension for Robot Framework

## Install

`:CocInstall coc-robotframework`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-robotframework-keymap)`

## Lists

`:CocList demo_list`
